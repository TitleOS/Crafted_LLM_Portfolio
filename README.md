## 💡 Featured Methodologies

### The "Spark" Pipeline: Synthetic Distillation
My **Spark-270m** model represents a successful application of the Microsoft Phi "Teacher-Student" training method.
1.  **Teacher Configuration:** I utilized **Lightning-1.7B** (my custom Qwen3 fine-tune) as the teacher model. Because Lightning was trained on the Hermes-3 dataset, it possesses a unique "unhinged" creativity and high steerability.
2.  **Synthetic Generation:** Lightning generated a corpus of synthetic "textbooks"—highly dense, educational, and structured data—specifically tailored to transfer its knowledge to a smaller context.
3.  **Distillation:** The **Spark-270m** (Gemma 3) model was trained on this synthetic data. The result is a "Micro Utility" model that punches above its weight class, capable of high-density information processing (like title summarization and search query generation) while retaining the "fun" personality traces of its teacher.

### HomePhi & HomeGem: Local AI for Home Assistant
A major focus of my recent work is enabling high-quality, privacy-focused AI for home automation. 
* **HomePhi4** leverages the reasoning capabilities of Phi-4-mini to understand complex user intents.
* **HomeGem** provides a lightweight conversational interface based on Gemma.
Both projects utilize the `acon96/Home-Assistant-Requests` dataset to ensure models understand specific smart home commands.

## 💻 Usage

You can load most of these models using the standard `transformers` library:

```python
from transformers import AutoTokenizer, AutoModelForCausalLM
import torch

# Example: Loading Lightning-1.7B
model_id = "TitleOS/Lightning-1.7B-Q8_0-GGUF" # Or the full FP16 path

tokenizer = AutoTokenizer.from_pretrained(model_id)
model = AutoModelForCausalLM.from_pretrained(
    model_id,
    torch_dtype=torch.float16,
    device_map="auto"
)

input_text = "Summarize the following text into a concise title:"
input_ids = tokenizer(input_text, return_tensors="pt").to("cuda")

outputs = model.generate(**input_ids, max_new_tokens=50)
print(tokenizer.decode(outputs[0]))
```
